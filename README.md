Career Recommendation System for Students

A web-based application that helps students discover suitable career paths based on their skills, interests, and academic background. The system provides personalized recommendations and guidance to support better career decisions.

Features: 
<br>
Student login and profile management
<br>
Skill and interest-based career recommendations
<br>
Personalized career suggestions
<br>
Course recommendations for skill improvement
<br>
User-friendly interface
<br>

Technologies Used:
<br>
Frontend: HTML, CSS, JavaScript
<br>
Backend: Python
<br>
Framework: Flask
<br>
Database: SQLite / MySQL
<br>
Tools: VS Code, GitHub

This project aims to bridge the gap between student interests and future career opportunities through intelligent recommendations.
